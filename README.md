# BDD
Behavior Driven Development testing framework for the D programming language

# Documentation

[https://workhorsy.github.io/BDD/2.0.0/](https://workhorsy.github.io/BDD/2.0.0/)

# Generate documentation

```
dub --build=docs
```

# Run unit tests

```
dub test
```
